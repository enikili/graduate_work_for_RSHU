# AeroCast

AeroCast - дипломное веб-приложение на Go для получения прогноза погоды по городу, поселку, селу и другим населенным пунктам.

Проект показывает:

- текущие погодные условия;
- прогноз на ближайшие 1-3 дня;
- почасовой график вероятности осадков на 72 часа;
- историю последних запросов, сохраненную в SQLite.

## Что внутри

- `Go` как основной язык и HTTP-сервер;
- `Open-Meteo API` для геокодинга и прогноза погоды;
- `SQLite` для хранения локальной истории запросов;
- `HTML + CSS + JavaScript` для интерфейса с собственной визуальной стилистикой.

## Почему выбран такой стек

- не требуются API-ключи для учебного MVP;
- приложение легко запускать локально;
- проект удобно расширять до десктопной или мобильной версии;
- SQLite упрощает дипломную часть с описанием слоя хранения данных.

## Структура проекта

```text
.
|-- internal/
|   |-- api/        # клиент Open-Meteo
|   |-- store/      # SQLite и история запросов
|   `-- web/        # HTTP-обработчики
|-- static/         # стили и frontend-логика
|-- templates/      # HTML-шаблоны
|-- data/           # SQLite база создается при запуске
|-- main.go
`-- README.md
```

## Запуск

```bash
go mod tidy
go run .
```

После запуска приложение будет доступно по адресу:

```text
http://localhost:8080
```
